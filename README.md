## Configuração dos Ambientes

Caso queira executar localmente, rodar o NPM no diretório do projeto e instalar as dependências necessárias
 
```
npm install
```

ou se quiser utilizar o docker, existem duas formas de executar o projeto: 
    
docker-composer

```
docker-compose up --build
```

kubernetes.

```
kubctl apply -f app-javascript.yml
```
